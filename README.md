## MyPixelgram — клиентская часть командного fullstack-приложения.

Реализует пользовательский интерфейс для социальной платформы с постами, подписками, платежами и real-time уведомлениями.

## 🛠 Tech Stack

- **Framework:** Next.js, React, TypeScript  
- **State Management:** Redux Toolkit  
- **Data Fetching & Server State:** RTK Query  
- **API & Realtime:** REST API, WebSocket  
- **Forms & Validation:** React Hook Form, Zod  
- **Styling:** Tailwind CSS  
- **UI Development:** Storybook  
- **Additional Libraries:** Radix UI, react-easy-crop
